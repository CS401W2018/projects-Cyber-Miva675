  • What is the main difference between CSS Grid and CSS Flex?
    • -A CSS grid is two dimensional while the CSS flex is one dimensional. CSS grid allows the parent elements to give specific commands to the child elements in which they strictly obey.Often causing the content to overlap or be shorten if it is large in size. However, a CSS flex parent elements allows a content within to enlarge if necessaary due to the content having larger size.
