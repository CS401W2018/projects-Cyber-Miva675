The viewport meta tag is important in a html document while creating a webpage. It technically allows the webpage to automatically stretch when displayed on a laptop or a smartphone.

If you are to adjust the CSS file for printing, you simply copy everything in the CSS and paste it in a new css file with a different filename. However the CSS file can stay the same, while the new file can be edited to show how you want the page to be displayed before printing.

When your webpage does not appear to be relevant or displayable in a print view, you can simply adjust the newly created css to allow a clear display of the necessary parts of the webpage in a print view.
