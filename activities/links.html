An absolute link is basically the whole address to another website while a relative link is a link to a page within the same website. Using relative links can provide an easier way for the web designer to edit and much clearer for someone to view the other pages withing the same file or website.

3 Rules for a Relative link. -A relative link doesn't have a protocol and authority at the beginning. -The relative link's path refers to pages withing the same file or website. -A relative link has its path components, such as; .., /, to connect up or down the path.

Links do not always have to be in a NAV tag. It is basically for the major web page links to be easier when viewing a large part of a web page. Other smaller links can be outside the NAV tag.
